# Assets — תמונות וסרטונים להטמעה

מקום לכל המדיה שצריך להטמיע בלומדה.

```
assets/
├── images/   # תמונות: רקעים, אייקונים, hotspots, איורים
└── video/    # סרטונים להטמעה
```

## הנחיות
- שמות קבצים באנגלית, באותיות קטנות, עם מקפים: `hazard-spill-01.jpg`, `intro-narration.mp4`.
- תמונות: רצוי WebP/JPEG אופטימלי; שמרי על משקל נמוך לטעינה מהירה במובייל.
- סרטונים: H.264/AAC (mp4) לתאימות רחבה ב-LMS; כתוביות נפרדות (.vtt) אם נדרש.
- **רישום ב-manifest:** כל קובץ מדיה חייב להופיע תחת `<resource>` ב-imsmanifest.xml, אחרת המדיה לא תיטען ב-LMS.
- **חילוץ מ-PDF/מצגת:** כשהמקור הוא PDF או מצגת, התמונות מוטמעות בתוכו וצריך **לחלץ** אותן לכאן כצעד בנייה מפורש (למשל `pdfimages`/`pdftoppm` ל-PDF, או פתיחת ה-`.pptx` כ-zip וקריאת `ppt/media/`). אם החילוץ נכשל — לדווח למשתמשת אילו תמונות חסרות ומה לספק, ולא לשלוח hotspot ריק.
